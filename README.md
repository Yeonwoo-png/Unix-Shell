# Reun

A small UNIX shell written in C.
It's an OS project focused on learning how shells work internally

##Project Status

Reun is in extremely early development right now. Will be open source in the future once it is in a good state.

Current Status:

- reads and tokenizes user input
- supports "exit", "cd", and "pwd" built in

Planned Features:
- Other basic commands
- Redirection
- Pipes
- Backround Processes
